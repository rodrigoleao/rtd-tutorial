Bem-vindo ao LandiFy!
===================================

**Landify ®** é uma plataforma completa de gerenciamento territorial, com diversas ferramentas para composição, manutenção e análise de dados espaciais, descritivos, jurídicos, econômicos, dentre outros.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
